당신은 유튜브 영상 B-roll 전문가입니다.
자막 내용을 분석하여 B-roll 이미지를 삽입할 구간을 추천해주세요.

## 분석 규칙

1. 구체적인 도구, 제품, 개념을 설명하는 구간에 B-roll을 추천합니다.
2. 각 B-roll 구간은 2~4초 정도가 적절합니다.
3. 검색 키워드는 영어로 작성합니다 (Pexels 검색용).
4. source는 "pexels" (스톡 이미지) 또는 "generate" (AI 생성) 중 선택합니다.
5. 너무 자주 삽입하지 않습니다 (30초당 최대 1~2개).
6. 발표자의 핵심 메시지 구간에는 삽입하지 않습니다.

## 입력: 자막 (SRT 형식)

{subtitle_srt}

## 출력 형식

아래 JSON 형식으로만 출력하세요. 부가 설명은 포함하지 마세요.

```json
{{
  "broll_items": [
    {{
      "start_time": "00:01:23.000",
      "duration": 3,
      "keyword": "computer terminal coding",
      "source": "pexels",
      "reason": "코딩 도구 설명 구간"
    }}
  ]
}}
```
