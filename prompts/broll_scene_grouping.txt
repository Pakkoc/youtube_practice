You are a video editor analyzing a script to group consecutive paragraphs into visual "scenes".

Each scene shares the same background image. Consecutive paragraphs about the same topic/setting should be in one scene.

## Input
Paragraphs (numbered, first 100 characters each):
{paragraphs_text}

## Rules
1. Groups must contain CONSECUTIVE paragraph indices only (no skipping).
2. Each group should have 1~3 paragraphs. Prefer smaller groups (1~2) over larger ones. Single-paragraph groups are fine for distinct topics or transitions.
3. Every paragraph index must appear exactly once across all groups.
4. representative_index = the paragraph whose content best represents the scene visually.
5. Keep the total number of groups between {min_groups} and {max_groups}. Aim for the HIGHER end.
6. When in doubt, split into separate groups rather than merging.

## Output
Return ONLY a JSON array:
```json
[
  {{"group_id": 1, "paragraph_indices": [1, 2, 3], "representative_index": 2, "reason": "intro topic"}},
  {{"group_id": 2, "paragraph_indices": [4, 5], "representative_index": 4, "reason": "comparison section"}}
]
```
