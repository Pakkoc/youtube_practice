## 오늘 만들 것

대본(텍스트) 하나를 넣으면, AI가 **음성, 슬라이드, 배경 이미지**를 만들고 합성하여 **유튜브 영상**을 출력하는 시스템을 체험합니다.

```
script.txt (대본)
    ↓ Claude Code에 명령어 입력
    ↓ AI가 음성 + 슬라이드 + 배경 이미지 자동 생성
    ↓ 렌더링 + 합성
final_video.mp4 (완성 영상, 1920x1080)
```

대본 하나로 **3가지 콘텐츠**가 나옵니다:

| 포맷 | 비율 | 명령어 | 용도 |
|------|------|--------|------|
| 유튜브 영상 | 16:9 (1920x1080) | `/generate-video` | 유튜브 본편 |
| 쇼츠/릴스 | 9:16 (1080x1920) | `/generate-shorts` | 유튜브 쇼츠, 인스타 릴스 |
| 카드뉴스 | 4:5 (1080x1350) | `/generate-carousel` | 인스타그램 캐러셀 |

## 이 시스템은 어떻게 동작하나?

```
대본 (script.txt)
  ↓
[1] 문단 분리 — 빈 줄 기준으로 자동 분리
  ↓
[2] 세 가지가 동시에 만들어짐:
    ├── 음성 합성 (ElevenLabs API — AI 성우)
    ├── 슬라이드 생성 (Claude Code → Remotion)
    └── 배경 이미지 (Gemini API — AI 일러스트레이터)
  ↓
[3] 슬라이드 렌더링 — 코드가 영상 클립으로 변환
  ↓
[4] 영상 합성 — 모든 클립을 하나로 연결
  ↓
[5] 자막 생성 + 합성
  ↓
final_video.mp4 완성!
```

## API 모드란?

| 모드 | GPU 필요 | 사용하는 서비스 | 워크숍 사용 |
|------|:--------:|---------------|:----------:|
| **Full 모드** | O (고사양 GPU) | 로컬 AI 모델 | X |
| **API 모드** | X (노트북 OK) | 클라우드 API | **O** |

**워크숍에서는 API 모드를 사용합니다.** 여러분의 노트북으로 충분합니다.

### 사용하는 외부 서비스

| 서비스 | 역할 | 비유 |
|--------|------|------|
| **ElevenLabs** | 대본을 사람처럼 읽어주는 음성 합성 | AI 성우 |
| **Gemini (Google)** | 슬라이드 배경 이미지 생성 | AI 일러스트레이터 |
| **Claude Code** | 슬라이드 디자인 코드 작성 + 시스템 제어 | AI 디자이너 |
| **Remotion** | 디자인 코드를 영상/이미지로 변환 | 렌더링 기계 |
| **FFmpeg** | 영상 클립들을 합치고 자막 합성 | 영상 편집 도구 |
| **OpenAI Whisper** | 음성→텍스트 변환 (쇼츠에만 사용) | AI 속기사 |
