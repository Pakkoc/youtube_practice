# Pipeline Flow Details

## Common Flow (all script-based pipelines)

```
script.txt
  -> 1. Paragraph split (+ scene split)
  -> 2. TTS dictionary registration & pronunciation correction (enhance_tts_dictionary)
  -> 3. TTS generation (ElevenLabs)
  -> 4. [Format-specific intermediate processing]
  -> 5. TSX Freeform generation (Claude Code writes TSX directly using skills)
  -> 6. [Format-specific intermediate processing]
  -> 7. Remotion render -> final video
```

**Key point**: Pipeline code has NO slide generation logic. Claude Code generates TSX using `slide-generation` skill, and the pipeline only collects existing TSX for rendering.

**Skill pre-generation -> pipeline skip pattern**:
- `slide-generation` skill -> `slides/*.tsx` -> pipeline collects only
- `broll` skill -> `broll/prompts.json` -> B-roll API call skip
- `tts` skill -> `tts_dictionary.yaml` enhancement -> TTS dict API call skip

**E2E video generation**: Use `/generate-video` command. Internally loads `slide-generation` + `broll` + `tts` skills. Running pipeline without slides produces empty video.

## script_to_video (7 steps)

```
1.  Split paragraphs -> 1.5 TTS dictionary auto-enhance (skip if skill pre-enhanced)
2.  [3-way parallel] Collect existing TSX/PY | TTS | B-roll images
    * Avatar clips start in background
3.  Remotion slide rendering (TSX/Manim + duration + B-roll background)
4.  Video compositing -> 5. Subtitle generation (deterministic, no Whisper)
6.  Avatar overlay (wait for clips) -> 7. Subtitle burn -> final_video.mp4
```

Output chain: `video_raw.mp4` -> `video_with_avatar.mp4` -> `video_with_subtitles.mp4` -> `final_video.mp4`

## script_to_shorts (6 steps)

```
1.  Split paragraphs + scene split -> 1.5 TTS dictionary auto-enhance (skip if skill pre-enhanced)
2.  TTS generation (ElevenLabs)
3.  Whisper word timestamps -> align with original script
4.  Load hook titles (hook_titles.json, pre-generated by skill)
5.  Remotion ShortsSlotPool parallel rendering (using existing TSX)
6.  FFmpeg compositing -> final_shorts.mp4
```

## video_to_shorts (4 steps)

Whisper word timestamps -> LLM viral segment selection -> FFmpeg trim -> Remotion 1080x1920 render

## script_to_carousel (Freeform TSX)

Claude writes TSX directly via `/generate-carousel` skill -> `scripts/regenerate_carousel.py` renders. 1080x1350 (4:5).
