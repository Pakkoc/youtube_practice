# Carousel Copywriting Evaluation Rubric

`/carousel-copywriting` 스킬의 자가평가에 사용하는 상세 채점 기준.
각 메트릭 5점 만점, 총 40점.

---

## 1. 훅 임팩트 (Hook Impact)

첫 장이 3초 안에 스크롤을 멈추게 하는가.

| 점수 | 기준 | 예시 |
|------|------|------|
| **5** | "문제+궁금증+숫자" 공식 완벽 적용. 제목 ≤10자. 피드에서 즉시 눈에 띔. 떡밥 키워드 포함. | "캐러셀 실수 5가지" (9자, 문제+궁금+숫자) |
| **4** | 공식 3요소 중 2개 적용. 제목 ≤12자. | "AI 자동화 루틴 3가지" (10자, 궁금+숫자) |
| **3** | 공식 일부 적용. 제목 ≤15자. 스크롤 멈출 수는 있음. | "프롬프트 잘 쓰는 방법" (10자, 궁금만) |
| **2** | 공식 미적용이나 주제는 전달됨. 제목 15자 초과. | "Chat GPT를 활용한 업무 자동화 가이드" |
| **1** | 훅이 아님. 정보 전달형 제목. 호기심 유발 0. | "AI 도구 소개" |

**체크포인트:**
- [ ] "문제" 요소: 독자의 고통을 찌르는가?
- [ ] "궁금증" 요소: 답이 궁금해지는가?
- [ ] "숫자" 요소: 구체적 수치가 있는가?
- [ ] 글자 수: 10자 이내 달성?
- [ ] 떡밥 키워드: CTA에서 회수 가능한 키워드 포함?

## 2. 세컨드 전략 (Second Slide Strategy)

두 번째 장이 "두 번째 커버" 역할을 하는가.
인스타그램은 첫 장을 넘긴 사용자에게 두 번째 장을 메인으로 재노출한다.

| 점수 | 기준 | 예시 |
|------|------|------|
| **5** | 3유형 중 명확히 선택. 콘텐츠에 최적. 독립적으로도 스와이프 유도. | 리스트형 콘텐츠에 미리보기형 → "오늘 다룰 5가지" |
| **4** | 유형 선택 적절. 스와이프 동기는 있으나 최적은 아님. | 비교 콘텐츠에 미리보기형 (한판정리형이 더 적합했을 수 있음) |
| **3** | 유형은 선택했으나 실행 약함. 독립 카드로는 약함. | 미리보기형이나 내용이 모호 |
| **2** | 전략적 의도 없음. 그냥 본론 시작. | "첫 번째는..." 바로 진입 |
| **1** | 커버와 내용 중복. 재노출 시 가치 없음. | 커버 내용을 다른 말로 반복 |

**3유형 적합성 가이드:**

| 콘텐츠 유형 | 최적 세컨드 전략 |
|------------|----------------|
| 리스트/단계형 | 미리보기형 (이어질 N가지 미리 보여주기) |
| 오해/반전형 | 오해/자칭형 ("이건 X가 아니라 Y다") |
| 비교/종합형 | 한판정리형 (전체 요약 테이블/목차) |
| 스토리/사례형 | 오해/자칭형 또는 미리보기형 |

## 3. 정보 밀도 (Information Density)

한 장에 하나의 메시지. 3~5초 안에 읽히는가.

| 점수 | 기준 |
|------|------|
| **5** | 전 카드 "미니제목+본문" 구조. 본문 전부 ≤150자. 한 장 한 메시지 철저. |
| **4** | 90%+ 카드 준수. 1장만 약간 초과 (≤180자). |
| **3** | 70%+ 카드 준수. 2-3장 초과. 초과분도 ≤200자. |
| **2** | 절반 미만 준수. 여러 장에서 다중 메시지. |
| **1** | 대부분 과부하. 본문 200자+ 다수. 읽기 포기 유발. |

**정보 밀도 원칙:**
- 미니 제목: 카드의 핵심을 한 줄로
- 본문: 미니 제목을 뒷받침하는 설명 (150자 이내)
- 본문이 길어지면 2장으로 분할
- 시각 요소(아이콘, 차트, 배지)로 텍스트 대체 가능

## 4. 언어 경제성 (Language Economy)

카드뉴스는 1장 3초 매체. 모든 텍스트가 최대 임팩트로 압축되었는가.

| 점수 | 기준 |
|------|------|
| **5** | 완전 개조식. 불필요한 조사/접속사 0. 숫자/데이터 적극 활용. 모든 항목 ≤20자. |
| **4** | 90%+ 개조식. 숫자 활용. 1-2개 항목만 20자 초과. |
| **3** | 대부분 개조식이나 일부 서술체 혼재. 추상적 형용사 간헐적 사용. |
| **2** | 서술체 비중 높음. "~합니다" 다수. 숫자 대신 형용사. |
| **1** | 장황한 서술체. 보고서/블로그 톤. 카드뉴스에 부적합. |

**변환 패턴 체크:**
- [ ] "그리고", "또한", "~하며" 같은 접속 표현 제거?
- [ ] "많은", "다양한", "큰" → 구체적 숫자로 대체?
- [ ] "~될 수 있습니다", "~하게 됩니다" → "바로 적용", "즉시 반영"?
- [ ] Before/After, 화살표(→) 활용?

**AI 안티패턴 체크 (밋밋함/뻔함 제거):**
- [ ] "~하는 것이 중요합니다" → 결과/수치로 대체?
- [ ] "다양한/효과적인/강력한" 같은 빈 형용사 0개?
- [ ] 3장 연속 같은 문장 구조 없음? (기계적 반복감 제거)
- [ ] "~해 보세요" 대신 단정형("이렇게 바꿔라") 사용?
- [ ] 각 항목에 감정(손실/이득 프레임)이 있는가? (정보 나열 금지)
- [ ] "요약하자면", "결론적으로" 같은 뻔한 전환어 없음?

## 5. 감정 곡선 (Emotion Curve)

카드 전체가 공감→전환→증거→실천의 서사 흐름을 따르는가.

| 점수 | 기준 |
|------|------|
| **5** | 4단계 완벽 구현. 온도 변화 2회+. 피크 타이밍 적절 (카드 5-7). 감정 흐름이 자연스럽게 읽힘. |
| **4** | 4단계 구현. 온도 변화 있으나 1번. 흐름은 자연스러움. |
| **3** | 대략적 흐름 있으나 단조로움. 온도 변화 없이 일정. |
| **2** | 정보 나열. 감정 의도 불분명. 순서 바꿔도 차이 없음. |
| **1** | 감정 곡선 없음. 무작위 배치. |

**감정 곡선 템플릿:**
```
공감 (카드 2~3): 독자의 현재 상황/고민. ●○○○○ ~ ●●○○○
전환 (카드 3~4): "다른 방법이 있다." ●●●○○
증거 (카드 5~7): 데이터, 사례, 비교. ●●●●○ ~ ●●●●● (피크)
실천 (카드 N-1~N): 행동 유도 → CTA. ●●●○○ ~ ●●●●●
```

**체크포인트:**
- [ ] 온도가 최소 2번 변하는가? (상승→하강 또는 하강→상승)
- [ ] 같은 온도 3장 연속 없는가?
- [ ] 피크가 중반~후반에 위치하는가?
- [ ] CTA가 감정적 여운을 남기는가?

## 6. 스와이프 동기 (Swipe Motivation)

각 카드 끝에서 다음 카드가 궁금한가.

| 점수 | 기준 |
|------|------|
| **5** | 전 카드에 자연스러운 Bridge. 내용 자체로 "다음이 궁금하다" 유발. "다음 장에서..." 같은 직접 표현 없이 연결. |
| **4** | 80%+ 카드에 Bridge. 1-2장만 연결 약함. |
| **3** | 50-80% 카드에 Bridge. 나머지는 독립적. |
| **2** | 일부 카드만 연결. 대부분 독립적 정보 전달. |
| **1** | 카드간 단절. 순서 무관. 슬라이드쇼 느낌. |

**Bridge 패턴:**

| 패턴 | 설명 | 예시 |
|------|------|------|
| 질문→답 | 질문을 던지고 다음 장에서 답 | "왜 이런 일이?" → (다음) 원인 제시 |
| 문제→해결 | 문제 제기 후 해결책 | "이게 문제다" → "해결법은..." |
| 전제→결론 | 전제를 깔고 결론 유보 | "A가 사실이라면" → "B가 가능하다" |
| 예고→상세 | 전체를 예고하고 하나씩 전개 | "3가지가 있다" → "첫 번째는..." |
| 대비→반전 | 통념을 제시하고 반전 | "다들 이렇게 한다" → "하지만..." |

## 7. 떡밥 회수 (Payoff Closure)

커버에서 던진 약속이 CTA에서 회수되는가.

| 점수 | 기준 |
|------|------|
| **5** | 커버 핵심 키워드가 CTA에서 명시적으로 회수. 처음과 끝을 이으면 완결감. 중간 카드에서도 키워드 1-2회 자연 반복. |
| **4** | CTA에서 회수됨. 완결감 있음. 중간 반복은 없음. |
| **3** | CTA에서 부분 회수. 의미는 통하나 명시적이지 않음. |
| **2** | CTA가 커버와 무관한 일반적 행동 촉구. ("팔로우 해주세요") |
| **1** | 회수 없음. CTA가 전체 내용과 단절. |

**떡밥 회수 체크:**
- [ ] 커버 키워드가 무엇인가?
- [ ] CTA에서 그 키워드가 등장하는가?
- [ ] 처음(커버)과 끝(CTA)만 읽어도 하나의 완결된 메시지인가?
- [ ] 중간에 키워드가 자연스럽게 1-2회 반복되는가?

## 8. 일관성 (Consistency)

카드 간 텍스트 분량, 구조, 톤이 일관적인가.

| 점수 | 기준 |
|------|------|
| **5** | 전 카드 분량 편차 ±20% 이내. "미니제목+본문" 구조 통일. 톤/어미 일관. 디자인 의도 명확. |
| **4** | 대부분 일관. 1-2장만 구조 다름 (의도적 변화는 OK). |
| **3** | 70%+ 일관하나 일부 카드가 눈에 띄게 다름. |
| **2** | 절반 정도만 일관. 분량/구조 들쭉날쭉. |
| **1** | 카드마다 다른 형식. 통일감 없음. |

**일관성 체크:**
- [ ] 모든 카드의 미니 제목 길이가 비슷한가?
- [ ] 본문 분량이 ±20% 이내인가? (가장 긴 카드 vs 가장 짧은 카드)
- [ ] 어미가 통일되어 있는가? (개조식 vs 서술체 혼재 없는가?)
- [ ] 커버와 CTA를 제외한 본문 카드들의 구조가 동일한가?

## 종합 채점 가이드

### 채점 절차

1. **카드 테이블 완성 후** 8가지 메트릭 각각에 대해 점수 부여
2. 각 점수에 **1문장 근거** 기록 (점수의 이유)
3. 총점 합산 → 등급 판정
4. **3점 이하 메트릭**을 "약점"으로 표시, 해당 카드 번호 특정

### 등급 기준

| 총점 | 등급 | 판정 | 행동 |
|------|------|------|------|
| 35-40 | **A** | 퍼블리시 레디 | → `/generate-carousel` 진행 |
| 28-34 | **B** | 소폭 수정 | → 약점 카드만 수정 후 재평가 |
| 20-27 | **C** | 주요 수정 | → 서사 구조 재설계 필요 |
| ≤19 | **D** | 재작성 | → Step 2부터 재시작 |

### 카드별 상세 평가 (선택)

메트릭 단위가 아닌 카드 단위로 평가가 필요할 때:

| 카드 # | 밀도 OK? | 경제성 OK? | Bridge? | 온도 적절? | 판정 |
|--------|---------|-----------|---------|-----------|------|
| 1 | - | ✅ | → 2 | ●●●●○ | PASS |
| 2 | ✅ | ✅ | → 3 | ●●○○○ | PASS |
| ... | | | | | |
