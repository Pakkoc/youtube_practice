# Carousel Editorial Strategy Reference

캐러셀 카피라이팅과 TSX 생성에 공통으로 적용되는 에디토리얼 가이드라인.
`/carousel-copywriting`과 `/generate-carousel`에서 참조.

---

## 1. 훅 공식 (Cover Hook)

### 공식: "문제 + 궁금증 + 숫자"

| 요소 | 역할 | 예시 |
|------|------|------|
| 문제 | 독자의 고통을 찌름 | "캐러셀 조회수 망치는" |
| 궁금증 | 답이 궁금하게 | "치명적 실수" |
| 숫자 | 구체성 → 신뢰 | "5가지" |

### 훅 패턴

| 패턴 | 예시 | 효과 |
|------|------|------|
| **숫자 + 결과** | "월 500만원 자동화 루틴 3가지" | 구체성 → 신뢰 |
| **반전/도발** | "코딩 몰라도 앱 만드는 시대" | 상식 파괴 → 호기심 |
| **질문형** | "아직도 수동으로 하세요?" | 자기 대입 → 공감 |
| **비교/대비** | "8시간 → 30분으로 줄인 방법" | Before/After → 임팩트 |

- 커버 제목 **10자 이내** 목표 (최대 15자)
- 부제는 제목을 보완하되, 제목만으로 클릭 동기가 충분해야 함
- **핵심 키워드**를 커버에 심어두면 떡밥 회수(CTA)에 사용 가능

## 2. 감정 곡선 (Emotion Curve)

```
공감 → 전환 → 증거 → 실천

- 공감 (카드 2~3): 독자의 현재 상황/고민. 낮은 온도 (●○○○○ ~ ●●○○○)
- 전환 (카드 3~4): "다른 방법이 있다." 온도 상승 (●●●○○)
- 증거 (카드 5~7): 데이터, 사례, 비교. 최고 온도 (●●●●○ ~ ●●●●●)
- 실천 (카드 N-1~N): 행동 유도 → CTA (●●●○○ ~ ●●●●●)
```

Intensity 전체 곡선에서 **최소 2번 온도 변화** 필요. 같은 감정 온도 3장 연속 금지.

---

## 3. 징검다리 전략 (Bridge Strategy)

카드는 독립된 슬라이드가 아니라 **하나의 이야기**다. 각 카드의 끝이 다음 카드의 전제가 되어야 한다.

연결 패턴:
- **질문 → 답**: "왜 이런 일이?" → 다음 카드에서 원인 제시
- **문제 → 해결**: "이게 문제다" → "해결법은..."
- **전제 → 결론**: "A가 사실이라면" → "B가 가능하다"
- **예고 → 상세**: "3가지가 있다" → 첫 번째는...

텍스트에 직접 "다음 장에서..."를 쓰지 말 것 — 내용 자체로 자연스럽게 연결.

---

## 4. 떡밥 회수 (Payoff/Closure)

커버에서 던진 핵심 메시지를 **마지막 CTA 카드에서 회수**한다.

- Cover 제목의 핵심 키워드/약속을 CTA에서 다시 언급
- 예: Cover "월 500만원 자동화의 비밀" → CTA "이제 당신도 자동화를 시작할 차례"
- 독자가 처음과 끝을 이었을 때 **완결감**을 느껴야 함
- 중간 카드에서도 커버 키워드를 1~2회 자연스럽게 반복

---

## 5. 언어 경제성 (Language Economy)

카드뉴스는 **1장 3초** 안에 읽히는 매체. 모든 텍스트를 최대 임팩트로 압축.

| 원칙 | Bad | Good |
|------|-----|------|
| 조사/접속사 제거 | "그리고 성능이 향상됩니다" | "성능 2배 향상" |
| 숫자로 대체 | "매우 많은 시간 절약" | "하루 3시간 절약" |
| 능동형 개조식 | "사용될 수 있습니다" | "바로 적용 가능" |
| 핵심만 남기기 | "이 기능은 사용자가..." | "자동 분석 → 리포트 생성" |
| 대비로 강조 | "좋은 결과를 얻었다" | "Before: 8시간 → After: 30분" |

- **제목**: 10자 이내 목표, 최대 15자
- **본문 항목**: 20자 이내 목표
- **숫자/데이터 우선**: 추상적 형용사 대신 구체적 수치

## 6. AI 안티패턴 (필수 점검)

AI가 쓴 카피는 "밋밋하고 뻔하다." 아래 패턴이 보이면 즉시 제거.

| AI 냄새 패턴 | 문제 | 대안 |
|-------------|------|------|
| "~하는 것이 중요합니다" | 뻔한 당위성 | 결과/수치: "저장률 3배 차이" |
| "다양한/효과적인/강력한" | 무의미한 형용사 | 구체 사실: "7가지", "30분→3분" |
| 모든 항목 같은 구조 | 기계적 반복 | 2-3장마다 톤/구조 변주 |
| "~해 보세요" | 수동적 제안 | 단정: "이렇게 바꿔라" |
| 항목 나열만 | 감정 없음 | 손실/이득 프레임 부여 |
| "요약하자면" | 뻔한 마무리 | 날카로운 한 줄 |

**핵심**: 독자가 "사람이 직접 경험하고 쓴 글"이라고 느껴야 한다.
